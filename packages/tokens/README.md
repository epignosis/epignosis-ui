# @epignosis_llc/ui-tokens

Framework-agnostic design tokens for the Epignosis design system. Ships compiled JS + `.d.ts` + a CSS variables stylesheet.

See the root [`DESIGN_TOKENS.md`](../../DESIGN_TOKENS.md) for the full token reference.

## Install

```bash
pnpm add @epignosis_llc/ui-tokens
```

## Usage

```ts
import { colors, spacing } from "@epignosis_llc/ui-tokens";
import "@epignosis_llc/ui-tokens/tokens.css";
```
